What It Does <br>
-Upload one or more label images <br>
-Extracts text using OCR <br>
-Checks for: <br>
  -Brand-like text <br>
  -Alcohol content (% or proof) <br>
  -Net contents (mL or L) <br>
  -Government warning header <br>
  -Shows results in the browser <br>
  -Exports results to an Excel file
